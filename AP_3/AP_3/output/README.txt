Tangent/Normal and FD summary

1D: tangent y = f(x0) + f'(x0)(x - x0); normal vector n ~ (-f'(x0), 1).
2D: tangent plane z = f0 + fx(x - x0) + fy(y - y0); normal n ~ (-fx, -fy, 1).
FD 1D: forward/backward O(h), central O(h^2), five-point O(h^4).
FD 2D: central O(h^2) and five-point O(h^4) for partials.
